# Mobile Games Portfolio By Ihssane

This is a simple storefront (GitHub Pages) that lists published mobile games and contact info.

## Deploy (GitHub Pages)
1. Create a repo named **YOUR_USERNAME.github.io**
2. Upload these files to the root of the repo
3. Go to **Settings â†’ Pages**
4. Set **Branch: main** and **Folder: /(root)**

Your site will be available at:

## Update games
Edit `index.html` and update the links/images inside the "Published Games" section.
